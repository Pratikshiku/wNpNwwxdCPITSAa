## Seaside Gymnasium Management System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/4690e5c7129d47f48aa89d556f1c3e7a/ghb20250918) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Equipment
5. Sports Forum
6. Sports News
7. Personal Center
8. My Posts
9. Change Password
10. Personal Information
11. Equipment Information
12. Equipment Borrowing
13. Equipment Return
14. Team Check-in
15. Entry Registration
16. Exit Registration
17. Activity Booking

**Administrator Functions:**
1. Login
2. Change Password
3. Student Management
4. Equipment Management
5. Equipment Borrowing Management
6. Equipment Return Management
7. Equipment Category Management
8. Team Check-in Management
9. Entry Registration Management
10. Exit Registration Management
11. Activity Booking Management
12. Lighting Maintenance Management
13. Sports Forum
14. Sports News
15. Carousel Management
